# Secured Digital Bank‑Vault System
This system allows users to check their balance or withdraw money from a simple bank account database.
It uses functions and exception handling to keep all operations safe and predictable.


## Features
- Bank accounts stored in a dictionary
- Function check_balance() returns the current balance
- Function withdrawal_money() processes withdrawals
- Converts ID and amount to numbers
- Handles invalid input (ValueError)
- Checks for unknown IDs
- Checks for insufficient balance
- Menu‑based system running until the user types "stop" or chooses option 3


# How to use
Choose an option:
1 → View balance
2 → Withdraw money
3 → Close the system

Valid ID → balance shown or withdrawal processed.
Invalid ID or invalid input → clear error message.
Type "stop" or choose 3 to exit.



## Learning Purpose
Practice with:
- dictionaries as a data store
- functions
- menu systems
- input validation
- exception handling
- basic financial logic
